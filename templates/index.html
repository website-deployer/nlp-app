API Documentation

REST API Endpoint

POST /analyze
Content-Type: application/json
Request Format:
{ "text": "Your text to analyze" }
Response Format:
{ "basic_stats": { "Total Characters": int, "Total Words": int, "Total Sentences": int, "Average Word Length": float, "Average Sentence Length": float }, "sentiment": { "Polarity": float, // -1 to 1 "Subjectivity": float // 0 to 1 }, "word_freq": { "word": int, // frequency count ... }, "bigrams": { "('word1', 'word2')": int, // frequency count ... }, "pos_tags": { "TAG": int, // frequency count ... }, "word_freq_plot": "base64_encoded_image", "sentiment_plot": "base64_encoded_image" }

Python Integration

Using the TextAnalyzer Class

from text_analyzer import TextAnalyzer # Initialize analyzer analyzer = TextAnalyzer("Your text here") # Get basic statistics stats = analyzer.get_basic_stats() # Get sentiment analysis sentiment = analyzer.get_sentiment_analysis() # Get word frequency word_freq = analyzer.get_word_frequency(top_n=10) # Get n-grams bigrams = analyzer.get_ngrams(n=2, top_n=10) # Get POS tags pos_tags = analyzer.get_pos_tags()

Data Processing Pipeline

The text analysis pipeline includes the following steps:

  1. Text tokenization using NLTK's word_tokenize
  2. Sentence segmentation using NLTK's sent_tokenize
  3. Stop word removal and punctuation filtering
  4. Part-of-speech tagging using NLTK's averaged_perceptron_tagger
  5. Sentiment analysis using TextBlob
  6. N-gram generation and frequency counting
